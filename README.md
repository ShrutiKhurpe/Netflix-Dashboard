# Netflix-Dashboard

1. Project Title / Headline

Netflix Data Analysis Dashboard – Exploring Content Trends, Genres, and Global Distribution

2. Short Description

This project presents an interactive dashboard analyzing Netflix movies and TV shows data.
It provides insights into content distribution, genre popularity, ratings classification, and global production trends.
The dashboard helps users understand how Netflix content has evolved over time.

3. Tech Stack
- Tableau (Data Visualization)
- Python (Data Cleaning – optional)
- CSV / Excel (Dataset)

4. Data Source
- Dataset: Netflix Movies and TV Shows Dataset
- Platform: Kaggle
Data includes:
- Title
- Genre
- Type (Movie / TV Show)
- Country
- Release Year
- Rating

5. Features / Highlights

i. Business Problem

Netflix has a large content library, making it difficult to analyze patterns manually.

There is a need to understand:
- Which genres are most common
- Which countries produce the most content
- Distribution of Movies vs TV Shows
- Trends in content release over time

ii. Goal of the Dashboard
- To provide a clear and interactive overview of Netflix content
- To identify trends in genres, ratings, and release years
- To compare Movies and TV Shows distribution
- To support data-driven insights

iv. Business Impact and Insights
- Movies dominate the platform, indicating a focus on widely consumable content
- The United States is the leading content producer
- Rapid growth after 2015 reflects Netflix expansion strategy
- Mature audience content (TV-MA, TV-14) is highly prevalent
- Drama and Documentary genres are the most common
